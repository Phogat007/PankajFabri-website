# Pankaj Fabricators — Website

Official website for **Pankaj Fabricators**, a precision metal fabrication company based in Jaipur, India, operating since 1978.

## Tech Stack

- **Next.js 15** — App Router, static prerendering
- **TypeScript**
- **Tailwind CSS** — Custom glassmorphism design system
- **Framer Motion** — Page animations
- **GSAP** — Scroll-triggered animations
- **next-themes** — Dark / Light mode

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — Hero, About, Products, Services, Values, Recent Works, CTA |
| `/about` | Company history, timeline, leadership |
| `/products` | Filterable product catalogue |
| `/services` | Service details and process |
| `/clients` | Client list, testimonials, sectors |
| `/contact` | Quote request form |

## Getting Started

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Production build
npm run build
```
